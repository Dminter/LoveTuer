# LoveTuer
基于[兔耳日记官网](http://www.tuer.me/)的第三方Android客户端实现
##由于此项目未维护多日(不好意思) 此版本可能无法运行,仅可参考代码 兔耳日记是个小清新非常不错的日记网站 开源Android端旨在有识之士能把app做好 更好服务用户
-base on [兔耳API](https://github.com/xiaojue/tuer.me/blob/master/docs/index.md)
